Trujillo Camping Services Site Plan

Site Name

The website's name will be "Trujillo Camping Services".

This name was selected because it easily communicates the type of content that the page will have which is camping services for areas in Trujillo.

Site Purpose

The site will provide information, equipment, and reservations services for camping spaces in Trujillo. It aims to help people plan beforehand and ensure proper preparation to get the best experience they could have in their outdoors adventure.

Scenarios

  • What are some campsites I could visit?
  • Which equipment should I get?
  • How can I know if the campsite is occupied?

Color Schema

The following color scheme will be used on the website:

  • #3F77AB Steelblue - For the header, the footer, and a few specific elements in the content for contrasting effect.
  • #7feedb Turquoise - For styling the elements in the content of the website, such as headings, cards, backgrounds, etc.

Typography

"Roboto" was chosen as the main font for clarity, consistency, and readability. It will be used as the primary font for the entire website, including for elements like headings, paragraphs, links, etc.
